version alfa 0.1

Как пользоваться:
Для входа использовать dev mode, кнопка "Вход по vk" не работает

Начальный экран входа `index.html`
Dev mode:
Пароль разработчика
vk id <-- пока любое
Имя пользователя

Уже есть семья? --да-> сразу перебрасывает к задачам `tasks.html`
 |
нет
 ↓
Выбор создать/по коду `choose.html`

Создать семью `create.html`
Введите имя вашей семьи → Семья создана `created.html` → перейти к задачам `tasks.html`

ИЛИ

Войти по коду `join.html` 
Введите код → Вы присоединились `joined.html` → перейти к задачам `tasks.html`

В `tasks.html` можно создать и удалить задачу, а также просматривать мои и все задачи. В левом верхнем углу можно посмотреть название семьи

Создание задачи `new_tasks.html`
Текст задачи
Кто исполнитель <-- можно выбрать себя (по умолчанию) или члена семьи
Дата <-- необязательна, по умолчанию не указывается

Профиль `profile.html`
В профиле отображается имя пользователя, а также список участников с ролью. По умолчанию присоединившийся по коду получает роль "Ребенок". Существуют роли "Создатель" - пользователь, создавший семью; "Ребенок" - вступивший по коду и роль "Родитель" - пользователь, обладающий правами "Создателя". Роль "Родитель" еще не реализована

Настройки `settings.html`
Уведомления <-- не реализовано
Код семьи (сгенерированный в `created.html`)
Роль в семье <-- пока статична 
Выйти <-- выход из сессии
